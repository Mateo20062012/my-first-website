Free Deployment Workflow

Create a GitHub Account: Sign up at GitHub.com and create a new "Repository."
Push from VS Code: Use the built-in "Source Control" tab in VS Code to sync your local folder with GitHub.
Connect to Vercel: Log in to Vercel with GitHub and import your new repository.
Auto-Sync: From now on, every time you "Sync Changes" in VS Code, your site updates automatically.
